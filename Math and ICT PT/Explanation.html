Definition of Relation and Function
(And Examples)

Sets:

Sets are groups of precisely defined objects like numbers.
The process of grouping objects into a set helps to set them apart from the components of other sets.
Here's Some Example For Sets!

Mapping:

Mapping applies to any set, it is a collection of numbers, shapes, etc. but in these mapping diagrams numbers are used. To indicate the link/relationship between the numbers, a line is drawn and the numbers are in the inside of an oval or circle.
Here's Some Example For Mapping!

Tables:

A table of values is a collection of ordered pair values that are typically produced by changing the values in an equation (relation).
A table can be horizontal or vertical.
